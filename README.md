# analyzer
## How to build

### Set up environment
*	sudo apt install cmake
*	sudo apt install git
*	sudo apt install build-esential
*	sudo apt-get install libpcap-dev
*	sudo apt install net-tools

### Clone this
* cd my_favorite_dir
* git clone https://github.com/KuzloPavlo/analyzer.git

## Build 
* mkdir build
* cd build
* cmake ../analyzer/ .
* cmake --build .

## Run
* cd ./analyzer/
* sudo ./analyzer
